HTML Form cheatsheet

HTML FORM REFRENCE

Form tags

Tag Name Description
<form> Form create a form container that wraps all the form elements
<input> Input create various types of input fields, such as text input, checkboxes etc..
<textarea> Textarea create a multi-line text input field, allowing users to enter larger amounts of text.
<select> Select create a dropdown list of options.
<button> Button create a button within a form.
<label> Label associate a text label with an input field
<fieldset> Fieldset group related form elements together.
<legend> Legend Used within a <fieldset> to provide a title or caption for the group of form elements.
<optgroup> Optgroup Used within a <select> element to group related options together.
<option> Option Used within a <datalist> element to definr a option.

Table Attributes

Attribute Name Description
action Action The URL or file where the form data should be submitted.
method Method The HTTP method to be used when submitting the form (e.g., GET or POST).
type Type The type of input field (e.g., text, checkbox, radio, password, etc.).
name Name A name for the input field, which is used to identify the field when the form is submitted.
value Value The initial value of thr input field
required Required the input field must be filled out before submitting the form.
